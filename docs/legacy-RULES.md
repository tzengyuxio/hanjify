# Hanjify 漢字轉換規則

本文件整理 Hanjify 專案中英文轉漢字的規則與設計原則。

## 一、設計理念

Hanjify 的核心概念是「反向羅馬拼音」——用漢字來表達英文，讓懂漢字但不懂英文的人也能猜測詞意。因此在選字上並非單純翻譯成中文，而是選擇能「表達英文原意」的漢字組合。

### 選詞考量

1. 共通通用均有的
2. 音近的
3. 從其他詞拼湊來的

### 取字考量（未排序）

- 是否符合本意或符合古字
- 是否較簡（結構簡單，例如都是既有結構，筆畫數不重要）
- 是否通用元件
- 順筆與否
- 是否既有俗字

## 二、特殊標記

| 標記 | 用途 | 範例 |
|------|------|------|
| `-` | 功能詞或單音節短詞，不需轉換直接保留原文 | a, and, as, at, be, by, do, go, in, no, of, on, or, to, the, is, it |
| 空白 | 尚未填寫，待處理 | — |

## 三、詞尾轉換規則

### 3.1 表示「人」的詞尾

| 英文詞尾 | 漢字 | 說明 | 範例 |
|----------|------|------|------|
| `-er` | `…者` | 實施動作的人 | actor→演者, reader→讀者, singer→歌者, player→遊者, printer→印者 |
| `-or` | `…者` | 執行動作的人 | survivor→生存者, producer→生產者 |
| `-er`, `-or` | `…家` | 專業人士 | artist→芸術家, scientist→科斈家, chemist→化斈家, writer→作家 |
| `-er`, `-or` | `…員` | 從事某職業的人 | assistant→協作員, member→成員, clerk→店員 |
| `-ist` | `…家` | 專業人員 | novelist→小說家, politician→政治家 |
| `-ician` | `…家` | 專業人員 | magician→魔法家, musicians→音楽家 |
| `-man` | `…人` | 從事某職業的人 | — |

**漢字詞尾選擇原則：**

- `～人`：某種類型、身分的人（軍人、主持人）
- `～者`：人或事物的代稱（記者、作者）
- `～員`：從事某種職業的人、團體中的一分子（教員、會員）
- `～家`：學有專長或有專門技術的人（專家、科學家）
- `～師`：具有專門技藝的人（醫師、律師）

### 3.2 形容詞詞尾

| 英文詞尾 | 漢字 | 範例 |
|----------|------|------|
| `-ive`, `-al`, `-ful`, `-ous` | `…的` | active→活動的, agricultural→農業的, beautiful→美的, financial→財政的 |

### 3.3 副詞詞尾

| 英文詞尾 | 漢字 | 範例 |
|----------|------|------|
| `-ly` | `…地` | actually→实際地, naturally→自然地, newly→新地, strongly→強地 |

### 3.4 名詞詞尾

| 英文詞尾 | 漢字 | 範例 |
|----------|------|------|
| `-tion`, `-sion` | 依語意 | addition→加祘, action→行動 |
| `-ness` | 依語意 | — |
| `-ance`, `-ancy` | `…狀` | acceptance→認受狀 |
| `-thing` | `…物` | something→某物, anything→任物, nothing→无物, everything→每物 |

## 四、詞首轉換規則

| 英文詞首 | 漢字 | 範例 |
|----------|------|------|
| `any-` | `任…` | anybody→任体, anyone→任一, anything→任物, anywhere→任何処 |
| `un-` | `不…` / `未…` | unable→不能, unhappy→不喜, unknown→未明, unreasonable→不理可 |
| `re-` | 依語意（多為動詞重複） | repair→修理, remember→思出, renew→更新 |

### 7.2 不改動的字

| 字 | 原因 |
|----|------|
| 黑 | 因為墨字 |
| 曾/增 | 從田的曾無法顯示 |
| 動 | 理由未定 |
| 傳 | 暫不改 |
| 隊 | 暫不改 |
| 鄰 | 不簡（邻），不改(隣) |
| 圍 | 俗字譜之異體多為「韋」字下方從「巾」 |
| 勝 | 暫不改 |
| 華/畢 | 有偏旁類推疑慮 |
| 風 | 風作為很多字的偏旁，改會改很多 |
