#### M2d wordlist 整詞優先

整詞 wordlist 對應永遠優先於拆解規則：princess→王女、review→審查、return→帰、meeting→会議。這讓特殊／約定俗成詞義得以保留，規則只在 wordlist 未覆蓋時 fallback。

#### 派生後綴選字理由

- `-er → 者`（不細分師／員／家）— 採「結構對應」原則，不做語意翻譯
- `-eer → 師`（新增）— engineer→工程師 等明顯為專業技藝
- `-ess → 者`（不標女性）— CJK 借字傳統不分陰性；個別詞如 princess→王女 由 wordlist 覆寫
- `re- → 再`（不採「重」）— 避免與「重要／重力」衝突，且符合 H2-B pan-CJK 中立
- `un- → 不`（不採「非」）— 避免與「非常 = very」嚴重衝突（wordlist 已有 unusual→非常 bug 例證）
- `non- → 非` — 與 un- 區分，表達範疇否定

### ✓ C4 [P2] when 連詞 vs 疑問詞的程式判斷 — 已決

**結論：採啟發式分流（C4-a）**

**判斷規則：**
- 句首大寫 `When` + 子句末 `?` → **何時**（疑問詞）
- 其他情境 → **当**（連詞）

**實作流程：**
1. 把每行依 `.!?` 切子句，記錄每個子句的結尾
2. `when` 出現時查所在子句結尾是否為 `?`
3. 是 → 何時；否 → 当

**覆蓋率：**

| 情境 | 範例 | 結果 |
|---|---|---|
| 直接疑問 | When did you arrive? | ✓ 何時 |
| 時間連詞 | When I was young, ... | ✓ 当 |
| 間接疑問 | I don't know when he came. | ✗ 当（誤判，可接受）|
| 中段疑問 | Tell me, when did you...? | ✗ 当（少見）|

主流情境（直接疑問、連詞）100% 正確；誤判情境讀起來尚可。

### ✓ C9 [P2] `-less` 後綴處理 — 已決

**結論：採 `-less` → `无…的`（C9-a）**

#### 規則

派生後綴 `-less`（lacking／without X）轉為「无 X 的」結構：
- useless → 无使用的
- careless → 无慎的
- harmless → 无害的
- hopeless → 无望的
- nameless → 无名的（既有 CJK 詞）
- countless → 无数的（既有 CJK 詞）
- endless → 无尽的（既有 CJK 詞，盡→尽 採 C2 字形）

- **語義精準**：`-less` = "lacking/without" 對應 「无 X」

- **吻合 CJK 既有詞**：无數／无名／无尽 為現成成語
- **無新增字形成本**：`无` 已在 C2 共通字採用清單

- 與 M2b 規則一致（單後綴 → 固定漢字組合）
- wordlist 整詞優先（regardless→无論的 已收）
